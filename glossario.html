Termos em Inglês no Filme "Piratas do Vale do Silício"
Termos do Filme
Apple I & II: Computadores iniciais desenvolvidos por Steve Jobs e Steve Wozniak, com o Apple II se destacando por sua popularidade e design.
Macintosh: Computador lançado em 1984 que revolucionou a forma de interagir com PCs, introduzindo uma interface gráfica.
Microsoft & MS-DOS: Fundada por Bill Gates, a Microsoft popularizou o Windows, e o MS-DOS foi seu sistema operacional inicial.
Windows: Sistema operacional gráfico da Microsoft, essencial para milhões de computadores.
Source Code & Software: O código-fonte é a base de todo software, fundamental no desenvolvimento de tecnologia no Vale.
Hardware & Operating System: O hardware são os componentes físicos do computador, enquanto o sistema operacional gerencia os recursos do computador.
Database: Sistema para armazenar dados organizados, essencial para empresas e aplicativos.
Cloud Computing: Computação em nuvem, permitindo o armazenamento remoto de dados e aplicativos acessíveis via internet.
Artificial Intelligence (AI) & Machine Learning: Inteligência artificial, usada em automação e assistentes virtuais, com aprendizado de máquina melhorando sistemas ao longo do tempo.
Algorithm: Conjunto de instruções que formam a base de todos os softwares.
Virus & Blockchain: Vírus são softwares maliciosos, enquanto blockchain é uma tecnologia usada para transações seguras.
Virtual Reality (VR) & Augmented Reality (AR): VR cria ambientes simulados para interação, e AR sobrepõe informações digitais ao mundo real.
Internet of Things (IoT): Conexão de dispositivos físicos à internet para compartilhar dados.
5G & Smart Device: A quinta geração de tecnologia móvel oferece maior velocidade e conexão, enquanto dispositivos inteligentes como smartphones e smartwatches se conectam à internet.
Termos da Documentação
Network: Rede de comunicação entre computadores, uma base fundamental em qualquer arquitetura de sistemas abertos.
Firewall: Sistema de segurança que monitora e controla o tráfego de rede para impedir acessos não autorizados, crucial para a segurança em OSA.
Encryption: Criptografia, técnica usada para proteger dados, essencial para manter a segurança das informações em sistemas abertos.
Cloud Storage: Armazenamento em nuvem, uma tecnologia de armazenamento remoto que facilita o acesso e a escalabilidade dos dados em arquiteturas abertas.
Database: Banco de dados, onde as informações são armazenadas e organizadas de maneira acessível, fundamental em OSA.
Internet: A infraestrutura de rede global que conecta dispositivos e sistemas, uma parte essencial em qualquer arquitetura de sistemas abertos.
Operating System: Sistemas operacionais que gerenciam o hardware e recursos de software de sistemas e dispositivos interconectados.
Machine Learning: Aprendizado de máquina, que pode ser usado para analisar grandes volumes de dados em sistemas abertos.
Algorithm: Algoritmos usados para processar dados e realizar funções específicas em sistemas de arquitetura aberta.
Source Code: O código-fonte, que deve ser acessível e bem estruturado para garantir a flexibilidade e modificação em sistemas abertos.
Programming Language: Linguagens de programação como C, Java, Python, usadas para criar sistemas interoperáveis.
